# 搜索统计与分析设计

> 搜索日志不只是调试数据，是用户画像和预设推荐的核心原料。

## 三层次洞察模型

### 1. 用户画像 → 从搜索习惯反推工作场景

```
高频关键词                → 推断场景         → 画像标签
─────────────────────────────────────────────────
"胶片" / "柯达" / "5213"  → 胶片模拟调色     → 影视后期 / 胶片爱好者
"709" / "log" / "REC709"  → 专业色彩空间意识  → 专业调色师 / 有基础
"电影感" / "cinematic"    → 电影感风格偏好    → 视频创作者
"冷色调" / "冷白"         → 冷调审美          → 时尚 / 商业广告
"动漫" / "动漫风"         → 二次元/ACG        → 动漫后期 / 游戏
"人像" / "肤色"           → 人像修图          → 摄影 / 婚纱
"繁花" / "仿色"           → 影视参考复刻      → 高级调色 / 研究型
"阿勒泰"                  → 自然风光          → 风光摄影 / 纪录片
```

画像不是静态标签，是随搜索量增长的动态聚类。

### 2. 热点缺口 — 被搜到的少，且真的不符合用户场景

```
判定逻辑:
  低频搜索 + 低点击/低采纳 + 语义分析确认不匹配场景
  → 正确排除，不是系统问题

示例:
  用户从不搜"人像" → "人像质感"、"人像橙调" 这些 LUT 被淹没是合理的不需要调整
```

### 3. 冷门潜力 — 被搜到的少，但符合用户场景

```
判定逻辑:
  低频搜索 + 但与高频关键词语义相关 + 从未出现在 Top-5 结果中
  → 系统曝光不足，应该提升

示例:
  用户常搜"胶片"，但"富士CN仿色"从未被搜到
  → 它不是不需要，是用户不知道有这个 LUT
  → 应该出现在"你可能还喜欢"的推荐里
```

**这是最有价值的洞察**——不是所有没被搜到的都是没用的，有些是用户不知道它们存在的隐藏好货。

## 数据分层策略

### 死数据（全量统计，从第一条开始）

```
用途: 总量指标，对外展示

统计项:
  - 总搜索次数
  - 每个 LUT 被命中的总次数（出现在 Top-5 结果中）
  - 从未出现在结果中的 LUT 列表
  - 历史搜索词云
  - 平均检索耗时
```

### 活数据（滑动窗口，从第 N 条开始分析）

```
N = 20 (可配置)

活窗口定义: 最近 N 条搜索记录

用途: 行为分析，画像更新，推荐生成

统计项:
  - 活窗口中高频关键词 Top-5 → 当前工作场景
  - 活窗口 vs 全量高频关键词的偏离度 → 场景是否在变化
  - 活窗口中命中但未被点击的 LUT → 弱信号，可能不感兴趣
  - 活窗口中与高频关键词相关但从未出现的 LUT → 冷门潜力推荐

活窗口的意义:
  用户刚接触系统时搜索不熟练（学习期）
  第 1-20 条可能是"试试看这个能不能找到"
  第 21 条开始反映真实的搜索习惯 → 画像应该从这里取
```

## 数据流

```
lut search → 查询 + 结果
    │
    ▼
search_log 表 (SQLite)      .lut_vectors/search.db
    │
    ├─ 死数据: 全量聚合查询
    │
    └─ 活窗口: 最近 N 条
         │
         ▼
    画像标签更新
         │
         ▼
    lut stats → 终端输出
    lut recommend → "你可能还喜欢"推荐
```

## CLI 接口预留

```bash
lut stats                   # 总览仪表盘
lut stats --top 10          # 热门搜索词
lut stats --cold            # 从未被搜到的 LUT
lut stats --hidden          # 活窗口中冷门但可能符合场景的 LUT
lut history                 # 最近 20 条记录
lut recommend               # 基于当前活窗口的推荐
```

## 实现优先级

| 阶段 | 内容 | 说明 |
|------|------|------|
| 立即 | `search_log` 表 + 写入 | 从第一次搜索开始积累死数据 |
| 立即 | `lut stats` 基础版 | 总次数 / 热门词 / 冷门预设 |
| 后续 | 活窗口分析 | 积累够 N 条后启用 |
| 后续 | `lut recommend` | 冷门潜力推荐 |
